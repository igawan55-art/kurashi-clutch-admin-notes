A versatile productivity suite for WP admins offering dashboard notes, page-specific memos, internal Wiki, and custom alerts.

== Description ==

kurashi-clutch Admin Notes and Wiki is an all-in-one management tool designed to streamline team communication and knowledge sharing within the WordPress admin area.

Key Features:
* Dashboard Memos: Create shared notes visible on the dashboard for team communication.
* Page-Specific Memos: Attach important notes directly to posts or pages, selectable by post type.
* Internal Wiki: A structured, hierarchical knowledge base (Wiki) for internal documentation.
* Admin Alerts: Create customizable, color-coded alert banners for important announcements, controllable by post type and screen.
* Drag-and-Drop Organization: Easily reorder your Wiki pages with an intuitive interface.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure your settings via the new "kurashi-clutch Admin Notes and Wiki" menu in the admin sidebar.

== Frequently Asked Questions ==

= Can I use the Wiki for internal documentation? =
Yes, the Wiki feature is designed specifically for that purpose, supporting hierarchical structures and status management (Publish, Draft, Archived).

= Can I limit who can edit the notes? =
Yes, within the settings, you can define the minimum capability required to edit dashboard notes, wiki pages, and admin alerts.
